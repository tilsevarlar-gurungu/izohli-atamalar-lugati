# Oʻzbekcha - Inglizcha - Ruscha Koʻrsatkich

Bu koʻrsatkich barcha atamalarni oʻzbekcha nomi boʻyicha alifbo tartibida keltiradi.

- **Qalin** atamalar bosh yoki ona atamani anglatadi.
- *Yotiq* atamalar aloqali atamalarni anglatadi.
- Oddiy uslubdagi (qalin ham emas, yotiq ham emas) atamalar bola darajasidagi atamalarni anglatadi.
