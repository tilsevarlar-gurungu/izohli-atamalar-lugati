# Qaytalagich

**Inglizcha:** Iterator<br>
**Ruscha:** Итератор<br>
**Soha:** Dasturlov

**Qaytalagich** — bu ketma-ketlikdagi elementlarga ularning ichki tuzilmasini oshkor qilmasdan, ularga birma-bir, tartibli kirishni taʼminlaydigan mavhumlashtirmadir. qaytalagichlar keyingi elementga oʻtish yoki yana element bor-yoʻqligini tekshirish kabi elementlar ustida yurish mantigʻini oʻzida jamlaydi. Shuning uchun chaqiruvchi bitik maʼlumotning qanday saqlanganini bilishi shart emas. Bundan tashqari, ayrim qaytalagichlar yalqov tarzda ishlashi mumkin, yaʼni ular aylanayotgan elementlar oldindan tayyorlanib, xotirada mavjud boʻlishi shart emas. Boshqacha qilib aytganda, ular navbatdagi element ustida faqat talab qilinganda ishlaydi. Bu ulkan yoki cheksiz ketma-ketliklarni samarali qayta ishlashda xotirani tejashga yordam beradi.

### Qaytalagich va aylangich farqi

Aylangich ma’lum bitik boʻlagi qanday takrorlanishini nazorat qilish tuzilmasi boʻlsa, qaytalagich ketma-ketlikdagi elementlar qanday ziyorat qilinishini va keyingi elementga qanday oʻtilishini aniqlovchi dizayn qolipidir.
