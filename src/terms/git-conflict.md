# Töqnaşuv

**Inglizça:** Conflict<br>
**Rusça:** Конфликт<br>
**Aloqali:** Git<br>
**Soha:** Dasturlov<br>

**Töqnaşuv** - bu ikki yoki undan ortiq dasturçi bitikning aynan bir qismini turliça özgartirganida va Git bu özgarişlarni qaysi biri töğri ekanligini öz-bajaruvli tarzda aniqlay olmaganida yuzaga keladigan holatdir. Odatda bu holat quyi oqimni yuqori oqim bilan qöşuv asnosida yoki siljituv amallari bajarilayotgan paytda sodir bölişi mumkin.

Bilimsanar fanida töqnaşuv bitikning izçilligini saqlaş uçun muhim tüşunçadir. Git tizimi bitikni kör-körona özgartirib yubormaslik uçun jarayonni tohtatadi va dasturçidan muammoni qölda hal qilişini söraydi. Dasturçi har ikkala özgarişni tekşirib, keraklisini qoldirişi yoki ularni umumlaştirişi lozim. Faqat töqnaşuv hal qilingandan sönggina qayd amalini yakunlaş mumkin.
