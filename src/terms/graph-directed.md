# Yönalişli Tör

**Inglizça:** Directed Graph<br>
**Rusça:** Направленный граф<br>
**Ota-atama:** Aloqali Tör<br>
**Soha:** Aloqali Tör<br>

**Yönalişli tör** — Tugunlar örtasidagi boğliqliklar aniq bir yönalişga ega bölgan aloqali tör turi. Bu tuzilmada qirralar bir tugundan boşqasiga yönaltirilgan böladi. Dasturlov jarayonida uşbu model vazifalarning bajariliş tartibi yoki malumotlarning bir tomonlama oqimini tasvirlaş uçun juda qölay. Masalan, internet sahifalarining bir-biriga bergan havolalari aynan şu model orqali ifodalanadi. Yönalişlar mavjudligi sababli, bu tizimda yurib çiquv amali faqat körsatilgan yönaliş böylab amalga oşiriladi.

```
 [A] ----> [B]
  ^         |
  |         |
  |         v
 [C] <---- [D]
```
