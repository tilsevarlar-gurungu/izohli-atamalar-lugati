# Tortuv

**Inglizça:** Pull<br>
**Rusça:** Извлечение слиянием<br>
**Aloqali:** Git<br>
**Soha:** Dasturlov<br>

**Tortuv** - bu olis dafinadagi (yuqori oqim) yangi özgartirişlarni yuklab oliş va ularni hozirgi işlab turgan mahalliy qölingizga darhol qöşuv (merge) amalidir. Bu buyruq aslida ikkita ketma-ket jarayonni birlaştiradi: avval malumotlarni yuklaydi (yuklov), söngra ularni sizning bitigingiz bilan birlaştiradi (qöşuv).

Dasturlovda tortuv amali loyihaning eng ohirgi nushasi bilan hamnafas bölişi uçun işlatiladi. Jamoada işlayotganingizda, boşqa dasturçilar ona qölga qöşgan yangiliklarni öz bilimsanaringizda oliş uçun muntazam ravişda tortuv amalini bajarib turişingiz lozim. Agar sizning mahalliy özgarişlaringiz bilan kelayotgan yangiliklar orasida farq katta bölsa, tizim töqnaşuv haqida ogohlantirişi mumkin.

```
[Olis Dafina] ---- (Yangi qaydlar) ----> [Bilimsanaringiz]
                               |
                        $ git pull (tortuv)
                               |
                 [Yuklov] + [Qöşuv] = Yangilangan bitik
```
