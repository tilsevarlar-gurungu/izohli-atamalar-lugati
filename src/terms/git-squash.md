# Yahlitlov

**Inglizça:** Squash<br>
**Rusça:** Сплющивание<br>
**Aloqali:** Git<br>
**Soha:** Dasturlov<br>

**Yahlitlov** - bu loyiha tarihidagi bir neça ketma-ket kelgan qaydlarni bitta yagona qaydga birlaştiriş amalidir. Dasturlovda bu usul asosan "işçi" yoki çala qaydlar bilan tölib ketgan loyiha tarihini tozalaş va tartibga soliş uçun işlatiladi.

Bilimsanar fanida yahlitlov göyo bir neça kiçik qutilarni bitta katta qutiga qaytadan joylaştirişga öhşaydi. Masalan, bitta vazifani bajarayotib siz "Hato tuzatildi", "Yana bir tuzatiş", "Imlo hato tuzatildi" kabi 3-4 ta mayda qaydlar yaratgan bölişingiz mumkin. Ularni asosiy loyihaga qöşişdan oldin yahlitlov orqali "Vazifa töliq bajarildi" degan bitta mazmunli qaydga aylantirasiz. Bu jarayon odatda interaktiv siljituv yordamida amalga oşiriladi.

```
[Tarih]: (Qayd 1) --- (Qayd 2) --- (Qayd 3)
           |           |           |
           +-----------+-----------+
                       v
[Yahlitlovdan söng]: (Yahlitlangan Qayd)
```
