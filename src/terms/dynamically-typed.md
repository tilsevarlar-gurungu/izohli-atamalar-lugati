# Beqaror-jinsli

**Inglizcha:** Dynamically-Typed<br>
**Ruscha:** Динамическая типизация<br>
**Soha:** Dasturlov

**Beqaror-jinsli** — bu dasturlov tillarining shunday xususiyatiki, bunda yorliqning ma’lumot jinsi tuzuv jarayonida emas, balki ijro vaqtida aniqlanadi. Bunday tillarda bitta yorliq avval sonni, keyinroq esa matnni saqlashi mumkin, chunki tuzgich yorliq uchun qat’iy jins talab qilmaydi. Bu xususiyat dasturchiga bitikni tezroq yozish va moslashuvchanlik imkonini beradi, biroq jinsi bilan bogʻliq xatolar faqat dastur ishga tushgandan keyingina ma’lum boʻladi. Python yoki JavaScript kabi tillar beqaror-jinsli tillar sirasiga kiradi.
