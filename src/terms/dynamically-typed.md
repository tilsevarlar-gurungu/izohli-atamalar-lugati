# Beqaror-jinsli

**Inglizça:** Dynamically-Typed<br>
**Rusça:** Динамическая типизация<br>
**Soha:** Dasturlov

**Beqaror-jinsli** — bu dasturlov tillarining şunday hususiyatiki, bunda yorliqning malumot jinsi tuzuv jarayonida emas, balki ijro vaqtida aniqlanadi. Bunday tillarda bitta yorliq avval sonni, keyinroq esa matnni saqlaşi mumkin, çunki tuzgiç yorliq uçun qatiy jins talab qilmaydi. Bu hususiyat dasturçiga bitikni tezroq yoziş va moslaşuvçanlik imkonini beradi, biroq jinsi bilan boğliq hatolar faqat dastur işga tuşgandan keyingina malum böladi. Python yoki JavaScript kabi tillar beqaror-jinsli tillar sirasiga kiradi.
