# Teruv hatosi

**Inglizça:** Typo<br>
**Rusça:** Опечатка<br>
**Soha:** Matn<br>

**Teruv hatosi** — bu matnni yoziş yoki teriş jarayonida etiborsizlik oqibatida yöl qöyiladigan hatolikdir. U odatda harflarning örni almaşib qolişi, biror harfning tuşib qolişi yoki ortiqça belgi yozilişi körinişida namoyon böladi. Bunday hatolik til qoidalarini bilmaslikdan emas, balki jarayonning tezligi yoki barmoqlarning harakatidagi noaniqlikdan kelib çiqadi. Teruv hatosi matnning umumiy sifatiga tasir qilsa-da, köpinça mantiqiy mazmunni butunlay özgartirib yubormaydi. Biroq, rasmiy hujjatlarda yoki muhim bitiklarda bunday kamçiliklar tuşunmovçiliklarga sabab bölişi mumkin. Uni bartaraf etiş uçun yozilgan matnni diqqat bilan qayta öqiş va tekşiriş talab etiladi.

```
  ASL SÖZ:      KITOB
                 | |
  HATOLIKLAR:   KTIOB  (Örin almaşuvi)
                KITO   (Tuşib qolişi)
                KITOBB (Ortiqça harf)
```
