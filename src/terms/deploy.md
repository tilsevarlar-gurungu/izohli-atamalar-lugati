# Işga soluv

**Inglizça:** Deploy<br>
**Rusça:** Развертывание (Деплой)<br>
**Sifat şakli:** işga sol<br>
**Soha:** Dasturlov<br>

**İşga soluv** — bu tayyor bölgan bitik yoki dasturni foydalanuvçilar foydalanişi mumkin bölgan muhitga (serverga yoki qurilmaga) ötkaziş va uni işga tuşiriş jarayonidir. İşga soluv bosqiçida dastur barça kerakli sozlamalar bilan taminlanadi va haqiqiy iş jarayoniga tayyorlanadi. Bu jarayon köpinça öz-bajaruvli bölib, doimiy kirişim (DK) tizimlarining mantiqiy davomi hisoblanadi. Muvaffaqiyatli işga soluvdan söng, yangi hosila yoki yangilanişlar olamşumul tarmoqda foydalanişga oçiq böladi.

```text
[Dafina]  ------>  [Tuzuv]  ------> [Tekşiruv]
    |                  |                 |
    V                  V                 V
[Qaydlar] --------> [İşga soluv] <--- [Tayyor]
                       |
                +--------------+
                |   Server /   |
                | Foydalanuvçi |
                +--------------+
```
