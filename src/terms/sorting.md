# Saralov

**Inglizça:** Sorting<br>
**Rusça:** Сортировка<br>
**Aloqali:** Alhorazma<br>
**Soha:** Malumot Töplami<br>

**Saralov** — malumotlar töplamini (masalan, tizma yoki ketma-ketlik) malum bir tartibda, odatda ösiş yoki kamayiş böyiça joylaştiriş jarayonidir. Bu jarayon alhorazma yordamida amalga oşiriladi va qidiruv işlarini tezlaştiriş uçun hizmat qiladi. Saralov usullari hilma-hil bölib, ularning samaradorligi işlatilayotgan bilimsanar hotira va ijro vaqti zahiralariga boğliq. Dasturlovda eng köp qöllaniladigan amallardan biri bölib, u tartibsiz bitik va malumotlarni tizimlaştiradi.

```text
  KIRITMA:  [3, 1, 4, 2]
             |  |
  1-bosqiç: [1, 3, 4, 2] (3 > 1, almaştirildi)
                   |  |
  2-bosqiç: [1, 3, 2, 4] (4 > 2, almaştirildi)
                |  |
  3-bosqiç: [1, 2, 3, 4] (3 > 2, almaştirildi)
             |
  ÇIQARMA:  [1, 2, 3, 4]
```
