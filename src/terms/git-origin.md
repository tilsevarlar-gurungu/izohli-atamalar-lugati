# Asl

**Inglizça:** Origin<br>
**Rusça:** Оригинал<br>
**Aloqali:** Git<br>
**Soha:** Dasturlov<br>

**Asl** - bu bilimsanar fanida loyihaning olis dafinasi uçun işlatiladigan fitriy nomdir. Dasturçi biror loyihani internetdagi manbadan nushalab olganida, Git tizimi öşa manzilga "asl" (origin) deb nom beradi.

Bu tüşunça dasturçiga olis dafina manzili (masalan, https://github.com/user/repo.git) bilan işlaş örniga, qisqa va tüşunarli nomdan foydalaniş imkonini beradi. Siz bitiklaringizni "asl"ga surayotganingizda (suruv), aslida ularni loyihaning asosiy manbasiga yuborgan bölasiz. Bir loyihada bir neça olis dafina bölişi mumkin, ammo eng asosiy manba deyarli har doim "asl" deb yuritiladi.

```
[Sizning Bilimsanaringiz]
          |
          | --- (Suruv / Push) ---> [Asl (Origin)]
          | <--- (Tortuv / Pull) --- GitHub/Serverdagi Dafina
```
