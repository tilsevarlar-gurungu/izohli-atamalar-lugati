# Yuqori oqim

**Inglizça:** Upstream<br>
**Rusça:** Вышестоящий поток<br>
**Aloqali:** Git<br>
**Soha:** Dasturlov<br>

**Yuqori oqim** - bu malumotlar olinadigan asosiy manba yoki yönaliş hisoblanadi. Dasturlovda, agar siz biror dafinadan nusha olgan bölsangiz, öşa nusha olingan joy sizga nisbatan yuqori oqim sanaladi. Masalan, loyihaning GitHub'dagi rasmiy nushasi har doim yuqori oqim deb yuritiladi.

Bu tüşunça juda muhim, çünki quyi oqimda işlayotgan dasturçilar har doim öz bitiklarini yuqori oqimga moslab turişlari lozim. Siz ona qöldan eng ohirgi özgartirişlarni tortib olganingizda, malumot yuqori oqimdan sizning mahalliy qölingizga qarab oqadi. Aksinça, sizning yangiliklaringiz asosiy loyiha egasi tomonidan qabul qilinib, ona qölga qöşilsa, sizning bitigingz yuqori oqimning bir qismiga aylanadi.

```
[Yuqori oqim (Upstream)] 
          ^         |
          |         | (Tortuv / Pull)
   (Suruv / Push)   |
          |         v
     [Quyi oqim (Downstream)]
```
