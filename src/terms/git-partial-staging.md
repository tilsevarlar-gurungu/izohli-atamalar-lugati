# Çala qaydlov

**Inglizça:** Partial Staging<br>
**Rusça:** Частичное индексирование<br>
**Aloqali:** Git<br>
**Soha:** Dasturlov<br>

**Çala qaydlov** - bu bitta fayl içidagi barça özgarişlarni emas, balki ularning faqat malum bir qismini (satrlarini yoki bölaklarini) qayd maydoniga ötkaziş jarayonidir. Bu usul dasturçiga bir neça turli mavzudagi özgarişlarni alohida-alohida va tartibli qayd qiliş imkonini beradi.

Bilimsanar fanida çala qaydlov — bitik tarihining "tozaligi" uçun juda muhim vositadir. Tasavvur qiling, siz bitta faylda hato tuzatdingiz va şuningdek, bitikni öqişni osonlaştiriş uçun biroz tahrir qildingiz. Ularni bitta qaydga qöşib yuboriş tavsiya etilmaydi. Çala qaydlov (`git add -p`) orqali siz avval hato tuzatilgan satrlarni saralab qayd qilasiz, söngra tahrir qilingan qismini alohida qayd qilasiz. Bu jarayon loyiha tarihini pala-partiş qaydlar bilan tölib ketişidan asraydi.

```
$ git add -p dastur.py
```
