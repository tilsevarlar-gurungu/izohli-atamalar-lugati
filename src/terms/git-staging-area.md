# Qayd maydoni

**Inglizça:** Staging Area<br>
**Rusça:** Область подготовки<br>
**Aloqali:** Git<br>
**Soha:** Dasturlov<br>

**Qayd maydoni** - bu bitikdagi özgarişlar rasman qayd qilinişidan oldin töplanadigan vaqtinça oraliq bosqiçdir.
Uni "tayyorgarlik maydoni" deb ham atasak böladi. Dasturlovçi tahrirdagi oğoçda köp özgarişlar qilişi mumkin,
lekin ulardan faqat malum bir qismini qaydga qöşiş uçun aynan şu maydonga ötkazadi.

Bilimsanar fanida bu qatlam "savatça" vazifasini bajaradi. Siz `git add` buyruği orqali özgarişlarni qayd
maydoniga joylaysiz. Bu hali bitik tarihiga muhrlanmagan, lekin "qayd qilişga tayyor" holatdir.
Bu maydonning mavjudligi dasturlovçiga öz özgarişlarini çala qayd qilmasdan, tartib bilan
va guruhlab muhrlaş imkonini beradi.

```
[Tahrirdagi oğoç] --(git add)--> [Qayd maydoni] --(git commit)--> [Dafina]
  (Fayl tahriri)                   (Saralaş)                      (Muhrlaş)
```
