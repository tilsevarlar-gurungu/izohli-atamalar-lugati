# Hotira sizuvi

**Inglizça:** Memory Leak<br>
**Rusça:** Утечка памяти<br>
**Soha:** Dasturlov<br>

**Hotira sizuvi** — dasturlov jarayonida işlatilgan hotira qismining endi kerak bölmasa-da, tizimga qaytarib berilmasligi holatidir. Bu muammo odatda yorliq yoki körsatkiçlar notöğri boşqarilişi natijasida kelib çiqadi. Hotira sizuvi uzoq vaqt davom etsa, bilimsanar zahiralarini tugatib, tizimning sekinlaşişiga yoki butkul töhtaşiga sabab böladi. Çiqindi yiğuvçi tizimi mavjud bölmagan quyi-qatlamli tillarda uşbu nuqsonni aniqlaş va tuzatiş dasturçi zimmasidadir. Rust kabi tillarda esa olib turuv tekşiruvçisi bunday holatlarning oldini olişga yordam beradi.
