# Yuklov

**Inglizça:** Fetch<br>
**Rusça:** Извлечение<br>
**Aloqali:** Git<br>
**Soha:** Dasturlov<br>

**Yuklov** - bu olisdagi dafinadagi (masalan, GitHub) barça yangilanişlarni, yangi qöllarni va qaydlarni mahalliy hotiraga köçirib oliş amali, biroq u sizning hozirgi işlab turgan bitigingizga (tahrirdagi oğoçga) tasir qilmaydi. Yuklov amali faqat "nima yangiliklar bor?" degan savolga javob oliş uçun hizmat qiladi.

Bilimsanar fanida yuklov va tortuv orasidagi farqni tüşuniş muhim: tortuv amali ham yangiliklarni yuklab oladi, ham bitigingizga qöşib yuboradi. Yuklov esa bitikni özgartirmasdan, faqatgina yuqori oqim va quyi oqim orasidagi farqni köriş imkonini beradi. Bu hatolarning va kutilmagan töqnaşuvlarning oldini olişda juda havfsiz usuldir.

```
[Olis (asl) Dafina (Origin)] 
       (Qayd D, E qöşildi)
              |
              | $ git fetch
              v
[Mahalliy Dafina (Local)]
```

Bu holatda sizning bitigingiz özgarmadi, lekin yangi qaydlar haqida malumot keldi, holos.
