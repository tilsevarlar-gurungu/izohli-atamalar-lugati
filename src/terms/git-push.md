# Suruv

**Inglizça:** Push<br>
**Rusça:** Отправка<br>
**Aloqali:** Git<br>
**Soha:** Dasturlov<br>

**Suruv** - bu mahalliy dafinada (sizning bilimsanaringizda) amalga oşirilgan va qayd etilgan özgartirişlarni olis dafinaga (masalan, GitHub'ga) yuboriş amalidir. Bu jarayon orqali sizning şahsiy bilimsanaringizdagi bitiklar yuqori oqim bilan qöşiladi va jamoaning boşqa azolari uçun ham oçiq va tayyor böladi.

Bilimsanar fanida suruv amali malum bir vazifa yakunlanganda va bitik muvaffaqiyatli qayd qilingandan söng bajariladi. Suruv amalini bajarayotganda, Git tizimi avval olis dafinada sizda yöq yangi özgarişlar bor-yöqligini tekşiradi. Agar yuqori oqimda yangi bitiklar bölsa, tizim sizdan avval tortuv qilişni, söngra suruv amalini bajarişni talab qiladi.

```
[Sizning bilimsanaringiz] --(Yangi Qaydlar)--> [GitHub/Olis Dafina]
          |                                          |
          +---------- $ git push (suruv) ----------->+
```
