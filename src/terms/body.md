# Tana

**Inglizça:** Body<br>
**Rusça:** Тело (Боди)<br>
**Soha:** Foydalanuvçi Yuzlanmasi<br>

**Tana** — bu sahifaning boşliq va adoq qismlari orasida joylaşgan asosiy mazmun maydonidir. Tana qismida foydalanuvçi uçun kerakli bölgan asosiy malumotlar, maqolalar, suratlar yoki boşqa interaktiv unsurlar jamlanadi. Foydalanuvçi tajribasi (FT) nuqtai nazaridan, tana eng köp diqqat markazida böladigan va eng köp özgaruvçan bölim hisoblanadi. Har qanday dastur yoki veb-sahifaning asosiy vazifasi aynan şu qatlam orqali amalga oşiriladi.

```text
+-----------------------+
|        Boşliq         |
+-----------------------+
|                       |
|       [ Tana ]        |
|   (Asosiy Mazmun)     |
|                       |
+-----------------------+
|         Adoq          |
+-----------------------+
```
