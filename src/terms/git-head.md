# Boş

**Inglizça:** Head<br>
**Rusça:** Указатель<br>
**Aloqali:** Git<br>
**Soha:** Dasturlov<br>

**Boş** - bu hozirgi vaqtda sizning bilimsanaringizda qaysi qayd yoki qöl ustida işlayotganingizni körsatuvçi
mahsus körsatkiçdir. Uni loyiha tarihidagi "siz şu yerdasiz" degan belgiga öhşatiş mumkin. Odatda,
boş siz tanlagan qölning eng ohirgi qaydiga yopişgan holatda turadi.

Bilimsanar fanida boşning örni juda muhim, çünki u yangi qaydlar qayerga qöşilini belgilaydi. Agar
siz biror-bir qayd yaratsangiz, Git uni aynan boş turgan nuqtaga ulaydi va boşni yangi qayd ustiga
suradi. Bazida dasturçi eski qaydlarni köriş uçun orqaga qaytganida, boş qöldan ajralib çiqişi mumkin,
bu holat esa uzilgan boş deb ataladi.
