# Tahrirdagi oğoç

**Inglizça:** Working Tree<br>
**Rusça:** Рабочее дерево<br>
**Aloqali:** Git<br>
**Soha:** Dasturlov<br>

**Tahrirdagi oğoç** - bu loyiha dafinasining sizning bilimsanaringizdagi fayl tizimida oçiq turgan holatidir. Bu siz bevosita bitik yozadigan, fayllarni öçiradigan yoki özgartiradigan "jonli" iş maydonidir. Siz fayllarni tahrir qilayotganingizda, bu özgarişlar hali Git'ning içki tizimiga (tarihiga) kirmagan böladi.

Bilimsanar fanida tahrirdagi oğoç va Git'ning içki oğoçi orasidagi boğliqlik şundaki, tahrirdagi oğoç — bu loyiha tarihining malum bir nuqtasidagi (odatda Boş turgan joydagi) fayllarning diskka çiqarilgan nushasidir. Siz bu yerda işni bitirib, özgarişlarni Qayd maydoniga ötkazmaguningizça, Git bu özgarişhlarni "sezmaydi".

```
[Git Ombori (.git)]  --- (Keltiruv / Checkout) --->  [Tahrirdagi oğoç]
 (Içki Oğoç)                                         (Sizning fayllaringiz)
                                                       ^
                                                       |
                                                Siz şu yerda işlaysiz
```
