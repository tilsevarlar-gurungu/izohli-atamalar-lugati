# Oğoç

**Inglizça:** Tree<br>
**Rusça:** Дерево<br>
**Aloqali:** Git<br>
**Soha:** Dasturlov<br>

**Oğoç** - bu loyihaning malum bir vaqtdagi holatini ifodalovçi içki malumot bölagidir. Git dafinasida har bir qayd bitta oğoçga işora qiladi. Bu oğoç öşa vaqtdagi barça fayllar va jildlarning qayerda joylaşgani va qanday nomlangani haqidagi "şajara"ni saqlaydi.

Dasturlov nuqtai nazaridan, bu oğoç — özgarmas tuzilmadir. Agar siz tahrirdagi oğoçda bitta harfni özgartirib qayd qilsangiz, Git yangi içki oğoç yaratadi. Şuning uçun ham Git juda tez işlaydi: u fayllarni şunçaki nushalamaydi, balki ularning oğoçsimon boğliqligini boşqaradi.

```
[Qayd (Commit)]
      |
      v
 [Oğoç (Tree)]  <-- (Loyiha şajarasi)
  /     \
Fayl_A   Jild_B
          \
           Fayl_C
```
