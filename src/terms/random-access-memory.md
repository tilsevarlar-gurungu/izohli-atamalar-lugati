# Tezkor Hotira

**Inglizça:** Random-Access Memory (RAM)<br>
**Rusça:** Оперативная память<br>
**Qisqartma:** TH<br>
**Soha:** Qurilma

**Tezkor hotira** — bu bilimsanar yoqilgan holatda malumotlarni vaqtinçalik saqlaş uçun hizmat qiluvçi öçar hotira turidir. U ijroçi uçun eng muhim zahira bölib, işlayotgan dastur bitiklari, faol yorliqlar va joriy topşiriqlar aynan şu joyda saqlanadi. Tezkor hotiraning asosiy hususiyati şundaki, undagi istalgan malumotni uning örin raqami orqali juda katta tezlikda öqiş mumkin. Biroq, quvvat uzilişi bilan uşbu hotiradagi barça malumotlar öçib ketadi, şuning uçun muhim malumotlar doimiy ravişda öçmas hotiraga yozib borilişi kerak.
