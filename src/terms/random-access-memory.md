# Tezkor Xotira

**Inglizcha:** Random-Access Memory (RAM)<br>
**Ruscha:** Оперативная память<br>
**Qisqartma:** TX<br>
**Soha:** Qurilma

**Tezkor xotira** — bu bilimsanar yoqilgan holatda ma’lumotlarni vaqtinchalik saqlash uchun xizmat qiluvchi oʻchar xotira turidir. U ijrochi uchun eng muhim zahira boʻlib, ishlayotgan dastur bitiklari, faol yorliqlar va joriy topshiriqlar aynan shu joyda saqlanadi. Tezkor xotiraning asosiy xususiyati shundaki, undagi istalgan ma’lumotni uning oʻrin raqami orqali juda katta tezlikda oʻqish mumkin. Biroq, quvvat uzilishi bilan ushbu xotiradagi barcha ma’lumotlar oʻchib ketadi, shuning uchun muhim ma’lumotlar doimiy ravishda oʻchmas xotiraga yozib borilishi kerak.
